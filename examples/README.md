# Examples

This directory contains example scripts demonstrating how to use Paper Collector.

## Available Examples

### `basic_usage.py`
Demonstrates basic programmatic usage:
- Parsing a PDF file
- Accessing paper metadata
- Filtering sections by type
- Exporting to JSON

**Usage:**
```bash
# Place a sample PDF in data/sample.pdf, then run:
python examples/basic_usage.py
```

## Future Examples

- `semantic_search.py`: Search papers using embeddings
